2-1. API 키 발급 및 입력

STT 변환과 AI 요약에 필요한 API 키를 설정합니다

앱 하단 설정 탭을 열고, 아래 순서대로 API 키를 입력합니다.

① 엔진 선택 (설정 탭 최상단)
엔진 유형권장 선택특징
STT 엔진CLOVA Speech한국어 특화, 화자 분리 지원
요약 엔진Gemini무료 한도 넉넉, 자동 모델 감지
② CLOVA Speech API (STT — 필수)
  1. console.ncloud.com 접속 → NAVER Cloud 계정 로그인
  2. Services → AI·NAVER API → CLOVA Speech
  3. "이용 신청" → 도메인 생성
  4. Invoke URL과 Secret Key를 복사
  5. 앱 설정 탭 → 해당 항목에 붙여넣기
⚠️ NAVER Cloud 계정이 없다면 먼저 가입이 필요합니다. 결제 수단 등록 후 이용 가능합니다.
③ Gemini API (AI 요약 — 권장)
  1. aistudio.google.com/apikey 접속 → Google 계정 로그인
  2. "Create API Key" 클릭 → 키 생성
  3. 생성된 키를 복사 → 앱 설정 탭에 붙여넣기
✅ Gemini API는 무료 한도 내에서 충분히 사용할 수 있습니다. 별도 결제 등록이 필요 없습니다.
④ Claude / ChatGPT (선택사항)
엔진API 키 발급비고
Claudeconsole.anthropic.com → API Keys유료
ChatGPTplatform.openai.com → API Keys유료
ℹ️ Gemini만 설정해도 요약 기능을 충분히 사용할 수 있습니다. Claude/ChatGPT는 선택사항입니다.

2-2. Google Drive 연동

녹음파일과 회의록을 자동으로 Google Drive에 업로드합니다

① Google Drive 연결
  1. 설정 탭 → Google Drive 섹션
  2. "Google Drive 연결" 버튼 탭
  3. Google 계정 선택 → 권한 허용
  4. 연결 상태가 "✅ 연결됨 (이메일)"으로 변경됨을 확인
② 업로드 폴더 설정
  1. "녹음파일 폴더 선택" → Drive 폴더 탐색 → 원하는 폴더 선택
  2. "회의록 폴더 선택" → Drive 폴더 탐색 → 원하는 폴더 선택
ℹ️ 녹음파일(.m4a)과 회의록(.md)을 각각 다른 폴더로 분리하여 업로드할 수 있습니다. 하위폴더까지 탐색 가능합니다.
③ 연결 실패 시
오류 메시지해결 방법
사용자가 로그인 취소다시 시도
Google Play Services 업데이트 필요Play Store에서 Google Play Services 업데이트
그 외 오류담당자(대표이사)에게 문의
✅ 초기 설정 완료 체크리스트
☐ CLOVA Speech Invoke URL + Secret Key 입력 완료
☐ Gemini API Key 입력 완료
☐ Google Drive 연결됨 표시
☐ 업로드 폴더 선택 완료
⚠️ Google Drive 연결은 선택사항입니다. 연결하지 않아도 녹음, STT, 요약, 공유 기능은 모두 정상 작동합니다.

3-1. 녹음하기 / 파일 불러오기

회의 음성을 직접 녹음하거나 기존 오디오 파일을 불러옵니다

① 실시간 녹음
  1. 하단 녹음 탭 → 빨간색 녹음 시작 버튼 탭
  2. 파일이름 입력 다이얼로그 → 이름 입력 (선택)
  3. 녹음 진행 중 화면에 경과 시간 표시
  4. 일시정지 → 재개 → 정지 가능
  5. 정지 시 M4A 파일 자동 저장
ℹ️ 백그라운드 녹음 지원: 앱을 닫거나 다른 앱을 사용해도 녹음이 계속됩니다. 상단 알림바에서 상태를 확인할 수 있습니다.
ℹ️ 통화 자동 감지: 녹음 중 전화가 오면 자동 일시정지, 통화 종료 후 자동 재개됩니다.
② 오디오 파일 불러오기
  1. 녹음 탭 → 파일 불러오기 버튼 탭
  2. 파일 선택기에서 오디오 파일 선택 (MP3, WAV, M4A 지원)
  3. 선택한 파일이 앱에 등록됩니다
③ 홈 위젯 (원터치 녹음)
  1. 홈 화면 길게 누르기 → 위젯 → "회의녹음요약" 검색
  2. 위젯을 홈 화면에 배치
  3. 위젯 탭 한 번으로 즉시 녹음 시작

3-2. STT 변환 및 AI 요약

녹음 파일을 텍스트로 변환하고 AI가 회의록을 작성합니다

① STT 변환 (음성→텍스트)
  1. 녹음 정지 후 또는 파일 선택 후 → STT 변환 버튼 탭
  2. CLOVA Speech 서버에서 음성→텍스트 변환 (화자 분리 포함)
  3. 변환 시간: 녹음 길이의 약 1/3 ~ 1/2 소요
  4. 변환 완료 시 텍스트가 화면에 표시됩니다
② AI 요약 (5가지 방식 선택)

STT 텍스트가 준비되면 요약 방식을 선택하고 요약을 실행합니다:

요약 방식설명추천 상황
화자 중심참석자별 발언 정리소규모 미팅, 면담
주제 중심안건별 내용 기반 요약여러 안건 논의 회의

강의 요약강의 노트 형태강의, 세미나, 교육

3-5. 자주 묻는 질문

Q. STT 변환이 너무 오래 걸립니다
A. CLOVA Speech는 서버에서 처리하므로 녹음 길이의 약 1/3~1/2 시간이 소요됩니다. 1시간 녹음 시 약 20~30분 소요됩니다. 인터넷 연결 상태를 확인하세요.
Q. 요약 결과가 만족스럽지 않습니다
A. 다른 요약 방식으로 다시 시도해보세요. 주제 중심은 안건별 정리에, 화자 중심은 소규모 미팅에 적합합니다. 요약 엔진을 변경(Gemini → Claude 등)해볼 수도 있습니다.
Q. 녹음 중 앱을 닫으면 녹음이 중단되나요?
A. 아닙니다. Foreground Service로 동작하므로 앱을 닫아도 녹음이 계속됩니다. 상단 알림바에서 녹음 상태를 확인할 수 있습니다.
Q. Google Drive 업로드가 되지 않습니다
A. 설정 탭에서 Drive 연결 상태가 "✅ 연결됨"인지 확인하세요. 업로드 폴더가 선택되어 있어야 합니다. 문제가 계속되면 담당자에게 문의하세요.
Q. API 비용이 얼마나 드나요?
A. Gemini API는 무료 한도 내에서 충분히 사용 가능합니다. CLOVA Speech는 NAVER Cloud 종량제로 과금되며, 월 300분 미만이면 소액입니다.
Q. 데이터가 외부로 유출되지 않나요?
A. 녹음 파일은 폰 로컬에 저장되고, STT/요약 시에만 API 서버로 전송됩니다. Google Drive 업로드는 본인 계정으로만 이루어집니다.
Q. 앱 업데이트는 어떻게 하나요?
A. 담당자로부터 새 APK 파일을 전달받아 동일하게 설치하면 됩니다. 기존 데이터(녹음, 회의록, API 키)는 유지됩니다.
Q. iPhone에서도 사용할 수 있나요?
A. 현재 Android 전용입니다. iOS 버전은 지원되지 않습니다.
