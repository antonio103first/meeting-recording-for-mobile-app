1-2. STEP 1 — 소스코드 다운로드

GitHub에서 최신 소스코드를 내 컴퓨터로 받아옵니다

https://github.com/antonio103first/meeting-recording-for-mobile-app

② Git Clone (권장)

PowerShell 또는 터미널에서 아래 명령어를 실행합니다:

cd C:\Users\[사용자명]\Documents git clone https://github.com/antonio103first/meeting-recording-for-mobile-app.git cd meeting-recording-for-mobile-app

  1. 위 주소로 접속합니다
  2. 페이지 오른쪽 상단의 초록색 Code 버튼을 클릭합니다
  3. Download ZIP을 선택합니다
  4. 다운로드 완료 후 ZIP 파일 압축을 해제합니다

1-3. STEP 2 — Firebase 설정 (google-services.json)

Google Drive 연동에 필요한 Firebase 설정 파일을 배치합니다

① Firebase Console 접속

  1. console.firebase.google.com 접속 → Google 계정 로그인
  2. 프로젝트 선택 (또는 새 프로젝트 생성)
  3. 프로젝트 설정 → 일반 → 내 앱 → Android 앱 추가
  4. 패키지 이름: com.krunventures.meetingrecorder 입력
  5. google-services.json 다운로드

② 파일 배치

다운로드한 google-services.json을 아래 경로에 복사합니다:

meeting-recording-for-mobile-app/ └── app/ └── google-services.json ← 여기에 배치

③ Google Cloud Console 추가 설정

  1. console.cloud.google.com 접속
  2. 같은 프로젝트 선택
  3. API 및 서비스 → 라이브러리 → Google Drive API 검색 → 사용 클릭
  4. API 및 서비스 → OAuth 동의 화면 → 구성 (앱 이름, 이메일 입력)
⚠ Google Drive API를 활성화하지 않으면 앱에서 Drive 로그인이 실패합니다.

1-5. STEP 4 — 스마트폰에 설치

빌드된 앱을 실제 폰에 설치합니다

1-7. 자주 묻는 질문 (설치편)

Q. iOS(아이폰)에서도 사용할 수 있나요?

A. 현재 Android 전용입니다. Android 8.0 이상이 필요합니다.

Q. 빌드 없이 APK만 받아서 설치할 수 있나요?

A. GitHub Releases에 APK가 제공되면 직접 설치할 수 있습니다. 폰 설정 → "출처를 알 수 없는 앱 설치" 허용이 필요합니다.

Q. Google Play Store에 있나요?

A. 현재 비공개 앱으로, Play Store에는 배포되지 않습니다.

Q. 업데이트는 어떻게 하나요?

A. GitHub에서 최신 코드를 pull 받고 다시 빌드 → Run 하면 됩니다.

Q. google-services.json 없이 빌드 가능한가요?

A. 빌드는 되지만 Google Drive 연동이 작동하지 않습니다.

2-3. STT 변환 & AI 요약

  • 액션 아이템 표 — No./담당자/실행과제/기한

2-5. Google Drive 연동

① 연결 방법

  1. 설정 탭 → 저장소/Drive 탭
  2. Google Drive 연결 버튼 탭
  3. Google 계정 로그인 → 권한 승인
  4. 연결 완료 → 자동으로 기본 폴더 생성

② 자동 업로드

"변환 완료 후 자동 업로드" 스위치를 ON 하면 STT 변환 + 요약 완료 후 자동으로 Google Drive에 업로드됩니다.

폴더저장 파일
회의녹음_음성파일.mp3 녹음 파일
회의녹음_회의록.txt (STT) + .md (요약)

2-6. 저장 폴더 설정 (SAF)

설정 탭 → 저장소 탭에서 파일 종류별 저장 폴더를 지정할 수 있습니다.

폴더 설정저장 파일
🎵 녹음파일 (MP3) 폴더.mp3 녹음 파일
📝 STT 변환 폴더.txt 변환 텍스트

2-7. 자주 묻는 질문 (사용편)

Q. 녹음 중 앱을 나가도 되나요?

A. 네, Foreground Service로 백그라운드 녹음이 유지됩니다. 상단 알림바에서 녹음 상태를 확인할 수 있습니다.

Q. 통화가 오면 녹음은 어떻게 되나요?

A. 통화 수신 시 자동 일시정지되고, 통화 종료 후 자동 재개됩니다.

Q. STT 변환에 얼마나 걸리나요?

A. CLOVA Speech 기준 30분 녹음 → 약 2~5분, 1시간 녹음 → 약 5~10분 소요됩니다.

Q. 카카오톡으로 회의록을 공유할 수 있나요?

A. 네, 회의 목록 → 공유 → "회의록 (PDF)" 형식을 선택하면 PDF로 변환되어 카카오톡에서 바로 볼 수 있습니다.

Q. 요약 양식을 바꾸고 싶어요.

A. 설정 탭 → 요약 방식에서 5가지 (화자/주제/공식/흐름/강의) 중 선택할 수 있습니다.

Q. Google Drive에 업로드가 안 됩니다.

A. Google Cloud Console에서 Google Drive API가 활성화되어 있는지 확인하세요. 또한 OAuth 동의 화면이 구성되어 있어야 합니다.

Q. 지정 폴더에 파일이 저장되지 않습니다.

A. 설정 → 저장소 탭에서 폴더를 다시 선택해주세요. Android 재시작 후 SAF 권한이 만료될 수 있습니다.

Q. 녹음 파일 크기는 얼마나 되나요?

A. M4A(AAC) 형식으로 1시간 녹음 시 약 30~50MB입니다. MP3 변환 후에도 비슷한 크기입니다.
